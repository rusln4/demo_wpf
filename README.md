# 👞 ShopApp — Система управления магазином обуви

![Avalonia UI](https://img.shields.io/badge/UI-Avalonia--UI-blue?style=for-the-badge&logo=dotnet)
![.NET 9](https://img.shields.io/badge/.NET-9.0-512BD4?style=for-the-badge&logo=dotnet)
![Entity Framework Core](https://img.shields.io/badge/DB-EF--Core-6236F5?style=for-the-badge&logo=sqlite)

Современное кроссплатформенное настольное приложение для управления ассортиментом товаров, заказами и пользователями магазина.

---

## ✨ Основные возможности

### 📦 Управление товарами
- **Интеллектуальный поиск**: Быстрый поиск по названию, описанию, производителю и категории.
- **Фильтрация и сортировка**: Удобные фильтры по поставщикам и сортировка по количеству на складе.
- **Визуальные индикаторы**: 
  - 🔵 **Синяя подсветка** для товаров, которых нет в наличии.
  - 🌿 **Зеленый фон** для товаров с высокой скидкой (>15%).
- **Редактирование**: Полноценный интерфейс для добавления и изменения характеристик товаров.

### 📝 Управление заказами
- **Создание и редактирование**: Удобный мастер оформления заказов.
- **Состав заказа**: Добавление различных товаров, изменение их количества и расчет итоговой стоимости в реальном времени.
- **Гибкая настройка**: Выбор пункта выдачи, назначение заказа на конкретного пользователя и управление статусами.
- **Валидация дат**: Поддержка заказов на 2025 и 2026 годы с автоматической проверкой корректности сроков доставки.

### 🔐 Безопасность и роли
- **Авторизация**: Система входа для пользователей с различными правами доступа.
- **Ролевая модель**: Разграничение функций между Гостями, Менеджерами и Администраторами.

---

## 🎨 Дизайн и интерфейс

Приложение разработано с акцентом на удобство и современный внешний вид:
- **Карточный дизайн**: Информация о товарах и заказах представлена в виде чистых, структурированных карточек.
- **Интуитивное управление**: Использование иконок, цветовых акцентов и продуманных отступов для повышения читаемости.
- **Кроссплатформенность**: Благодаря Avalonia UI приложение отлично выглядит на macOS, Windows и Linux.

---

## 🚀 Технологический стек

- **Framework**: [Avalonia UI](https://avaloniaui.net/)
- **Platform**: .NET 9.0
- **ORM**: Entity Framework Core
- **Database**: MySQL / MariaDB (с поддержкой миграций)
- **Patterns**: MVVM-lite подход, Service Layer

---

## 🛠️ Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/rusln4/demo_wpf.git
   cd demo_wpf
   ```

2. **Настройте базу данных**:
   Импортируйте файл `dump.sql` в вашу СУБД MySQL.

3. **Соберите и запустите**:
   ```bash
   dotnet build
   dotnet run --project ShopApp.csproj
   ```
